
#### NDK是什么

原生开发工具包 (NDK) 是一组可让您在 Android 应用中利用 C 和 C++ 代码的工具。 可用以从您自己的源代码构建，或者利用现有的预构建库

**NDK使用场景**

> NDK会不可避免的增加开发过程的复杂性,通常是不建议使用的.

- 从设备获取卓越性能以用于计算密集型应用，例如游戏或物理模拟。
- 重复使用您自己或其他开发者的 C 或 C++ 库。
